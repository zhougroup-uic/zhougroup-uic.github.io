Trajectory to PDB Converter

Converting MAEPPI trajectory files to PDB format.

Download source code

You can download the javascript code, maeppi_traj2pdb.js, here. To run, use the following command,

    nodejs maeppi_traj2pdb.js template.pdb trj --onlyCA --txt

where trj are config file from MAEPPI simulation.
